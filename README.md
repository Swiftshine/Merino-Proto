# Merino

A **prototype** Yoshi's Woolly World editor.

> [!WARNING]
> Merino is in its earliest stages of development. Things may be subject to change. Some features may not be present. Expect code to be very messy.

> [!NOTE]
> As Merino is still a prototype, **there will not be any official releases at this time.** To download a development version compiled on a commit-by-commit basis, go to the Actions tab. **You will need a GitHub account for this.**

## Controls (tentative)

Right-click to drag
Ctrl + scroll wheel to zoom
